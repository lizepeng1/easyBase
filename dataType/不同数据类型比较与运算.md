> 原始值与引用值
```js
1. 变量可以存放两种类型的值： 原始值 和 引用值
2. 原始值代表原始数据类型的值，也叫基本数据类型，包括 Number、String、Boolean、null、undefined。
3. 引用值指的是复合数据类型的值，包括 Object(Array也是Object)、Function、Date、RegExp。
```
> null == undefined

```js
1. 要比较相等性之前，不能将 null 和 undefined 转换成其他任何值，
2. 但要记住 null == undefined 会返回 true
```
> 数组的比较

```js
1. 数组执行tostring()方法后变成字符串，变成字符串的规则是值与值之间用"," 连接起来。
2. []空数组里边没有值会转为"",而""会转为数字0，数组中有一个值也是转为字符串，再转为数字
```
>对象的比较

```js
1. 会先执行valueof()的方法，如果得到的值不是原始值，会继续执行toString()的方法。
```
> +号运算符
```js
[]+{}  // "[object Object]"
{}+[]  // 0
+"5"   // 5
+[]    // 0

1. js会将第一个{}解释成了一个空的代码块并忽略了它。
2. +号运算符可以执行Number()操作。
```
> 两个值的比较规则
```js
1. 两边尽可能都转为number类型数据。
2. undefined、null、0、NaN、"", 在作为判断条件时都是false, 相比较时视情况而定。
3. null与undefined在与其他数相等运算时不进行类型转换。
4. NaN和任何数据做大小或等于比较永远返回false(包括NaN)。
5. 字符串与字符串相比不会转为Number类型，依次比较每一位字符的unicode编号。
6. 字符串与其他类型相比,都转为number类型。
7. 引用类型比较时, 遵循引用类型的规则, 先valueof(), 再toString()。
8. 其他情况都是false。
```
